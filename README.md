# Credit Scoring Model

## Objective
Predict an individual's creditworthiness using historical financial data using Machine Learning techniques.

## Features
- Data preprocessing
- Feature engineering
- Random Forest Classification
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
  - ROC-AUC

## Dataset Features
- person_age
- person_income
- person_home_ownership
- person_emp_length
- loan_intent
- loan_grade
- loan_amnt
- loan_int_rate
- loan_percent_income
- cb_person_default_on_file
- cb_preson_cred_hist_length

## Results
- Accuracy: 92.07%
- Precision: 96.43%
- Recall: 70.93%
- F1-Score: 81.74%
- ROC-AUC: 93.98%
